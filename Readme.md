Modüller İnince pm2 strat all Yazıp Botları Başlatıyoruz

                     =PM2 KOMUTLARI=

pm2 start all = botları başlatır

pm2 stop all = bütün botları durdurur

pm2restart all = bütün botları yeniden başlatır
